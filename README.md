# Descrição

Aplicação que faz o gerenciamento do faturamento pessoal, mostrando pagamentos que já foram realizados e os pendentes.

## Available Scripts

Entre no diretorio frontend:

### `npm start`

Roda no ambiente Dev.\
Abra [http://localhost:3000](http://localhost:3000) visualizar no navegador.


## Tecnologias

- React.js

- React Router DOM

- React Icons

- Typescript

- Sass

## Padrões

Foi utilizado o padrão CSS BEM para dar uma organizado nas folhas de estilo.
